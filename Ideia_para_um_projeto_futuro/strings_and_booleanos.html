Strings e Booleanos em JavaScript

Strings

Strings são cadeias de caracteres usadas para representar texto. Elas podem ser delimitadas por:

  • "aspas duplas"
  • 'aspas simples'
  • `crases` (Template Literals, que permitem interpolação e quebras de linha).

Características das Strings

  • Imutabilidade: Strings não podem ser alteradas após criadas. Alterações geram uma nova string.
  • Interpolação: Com template literals, podemos inserir valores diretamente em strings: 
    const nome = "Jean";
          const mensagem = `Bem-vindo, ${nome}!`;
          console.log(mensagem); // Bem-vindo, Jean!
  • Métodos úteis:
    • length: Retorna o tamanho da string.
    • toUpperCase() e toLowerCase(): Alteram o texto para maiúsculas ou minúsculas.
    • includes(substring): Verifica se a string contém outra string.
    • replace(antigo, novo): Substitui parte de uma string.
    • split(separador): Divide uma string em partes e retorna um array.

Exemplos Comparativos

// Comparando Strings
      const texto = "Olá, Mundo!";
      console.log(texto.length); // 12
      console.log(texto.toUpperCase()); // "OLÁ, MUNDO!"
      console.log(texto.includes("Mundo")); // true

Booleanos

Os valores booleanos representam apenas dois estados: verdadeiro (true) e falso (false).

Características

  • Comumente usados para controle de fluxo (ex.: em condições).
  • Podem ser gerados por comparações e operadores lógicos.

Operadores Comuns

  • ==: Igualdade de valor.
  • ===: Igualdade de valor e tipo (estritamente igual).
  • !=: Diferença de valor.
  • !==: Diferença de valor e tipo.
  • &&: E lógico (AND).
  • ||: Ou lógico (OR).
  • !: Negação lógica (NOT).

Valores Truthy e Falsy

Os valores booleanos também são inferidos a partir de outros tipos:

  • Falsy: false, 0, "", null, undefined, NaN.
  • Truthy: Todos os outros valores, como números diferentes de zero, strings não vazias, etc.

Exemplos

// Comparações
      console.log(10 > 5); // true
      console.log(10 === "10"); // false

      // Valores truthy e falsy
      const vazio = "";
      if (vazio) {
        console.log("Truthy");
      } else {
        console.log("Falsy");
      } // Saída: "Falsy"
