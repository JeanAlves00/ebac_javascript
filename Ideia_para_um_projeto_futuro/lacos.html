Laços em JavaScript

O que são Laços?

Laços (ou loops) permitem executar um bloco de código repetidamente enquanto uma condição é verdadeira. Eles ajudam a evitar repetição manual, tornando o código mais eficiente e legível.

Tipos de Laços em JavaScript

  • for: Um laço com número fixo de repetições, baseado em um contador.
  • while: Executa enquanto uma condição for verdadeira.
  • do...while: Executa pelo menos uma vez, e depois verifica a condição.
  • for...of: Itera sobre elementos de arrays ou strings.
  • for...in: Itera sobre as propriedades de objetos.

Laço for

O laço for é composto por três partes: inicialização, condição e incremento:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Saída: 0, 1, 2, 3, 4

Laço while

Repete o código enquanto a condição for verdadeira:

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Saída: 0, 1, 2, 3, 4

Laço do...while

Executa o código pelo menos uma vez antes de verificar a condição:

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

Saída: 0, 1, 2, 3, 4

Laço for...of

Usado para iterar sobre elementos de arrays ou strings:

const frutas = ["maçã", "banana", "laranja"];
for (const fruta of frutas) {
  console.log(fruta);
}

Saída: maçã, banana, laranja

Laço for...in

Usado para iterar sobre as propriedades de um objeto:

const pessoa = { nome: "Jean", idade: 25 };
for (const chave in pessoa) {
  console.log(chave, pessoa[chave]);
}

Saída:

  • nome Jean
  • idade 25

Quando usar cada laço?

  • for: Quando o número de iterações é conhecido.
  • while: Quando a condição depende de eventos externos.
  • do...while: Quando deseja garantir pelo menos uma execução.
  • for...of: Para iterar diretamente sobre arrays ou strings.
  • for...in: Para iterar sobre objetos.
