Funções em JavaScript

O que são Funções?

Funções em JavaScript são blocos de código que podem ser reutilizados. Elas recebem entradas (parâmetros), processam essas entradas e retornam uma saída.

Uma função pode ser definida uma vez e executada quantas vezes for necessário.

Tipos de Funções

  • Funções Declaradas: São carregadas antes da execução do código devido ao _hoisting_.
  • Expressões de Função: São atribuídas a variáveis e não sofrem _hoisting_.
  • Arrow Functions: Introduzidas no ES6, possuem uma sintaxe mais curta.
  • Funções Anônimas: Não possuem nome e são frequentemente usadas como _callbacks_.
  • Funções Assíncronas: Trabalham com operações assíncronas usando `async` e `await`.

Exemplos de Funções

Função Declarada

function soma(a, b) {
        return a + b;
      }
      console.log(soma(2, 3)); // 5

Expressão de Função

const subtrai = function(a, b) {
        return a - b;
      };
      console.log(subtrai(5, 2)); // 3

Arrow Function

const multiplica = (a, b) => a * b;
      console.log(multiplica(4, 5)); // 20

Função Assíncrona

async function buscaDados() {
        const resposta = await fetch('https://api.example.com');
        const dados = await resposta.json();
        console.log(dados);
      }

Conclusão

Funções são fundamentais no JavaScript, permitindo modularizar o código e torná-lo reutilizável e eficiente. Dominar as funções ajuda a escrever scripts mais claros e poderosos.
