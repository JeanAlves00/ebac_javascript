Objetos em JavaScript

O que são Objetos?

Objetos são uma estrutura fundamental em JavaScript, permitindo armazenar dados na forma de pares chave-valor. Eles são usados para representar coisas do mundo real, como um carro, um usuário ou configurações.

Como Criar Objetos

Os objetos podem ser criados de várias formas. Aqui estão algumas das mais comuns:

Usando Literais

const carro = {
        marca: "Toyota",
        modelo: "Corolla",
        ano: 2021
        };

Usando o Construtor

const pessoa = new Object();
pessoa.nome = "Jean";
pessoa.idade = 25;

Manipulando Objetos

É possível acessar, adicionar ou remover propriedades de objetos:

Acessando Propriedades

console.log(carro.marca); // Toyota
console.log(carro["modelo"]); // Corolla

Adicionando Propriedades

carro.cor = "Prata";

Removendo Propriedades

delete carro.ano;

Métodos em Objetos

Objetos podem conter funções como propriedades. Estas funções são chamadas de métodos.

const calculadora = {
    somar: function(a, b) {
    return a + b;
  },
  subtrair(a, b) {
    return a - b;
  }
};

console.log(calculadora.somar(5, 3)); // 8
console.log(calculadora.subtrair(9, 4)); // 5

Conclusão

Objetos são essenciais em JavaScript, ajudando a organizar dados e lógica de maneira eficiente. Dominar seu uso é um passo fundamental para se tornar um programador avançado.
